## [В ПРОЦЕССЕ] Этап 4: Интеграция ИИ (OpenRouter)
**Статус: Реализация сетевого слоя и логики оценки**

### Задачи:
1.  **Безопасность:** Настройка `BuildConfig` и `local.properties` для хранения ключа OpenRouter.
2.  **Сетевой слой:** Настройка Retrofit + OkHttp для работы с API OpenRouter (модель `google/gemma-3n-e2b-it:free`).
3.  **Кэширование:** Интеграция `AiCacheEntity` для сохранения результатов (0-100).
4.  **Логика:** Асинхронная оценка ТОП-10 рецептов через ИИ.
5.  **UI:** 
    *   Скелетон-загрузка во время ожидания ИИ.
    *   Синий цвет круга процентов для ИИ-оценки.
    *   Подпись "ИИ" под кругом оценки.
